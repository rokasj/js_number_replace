js_number_replace
================

Increment/decrement numbers found in a string.<br><br>
Inputs ('[]' - optional, '\n' - newline, '\t' - tab, not found characters need to be prefixed with '\'):<br>
1) [search_before](\d+)[search_after]<br>
2) [replace_before]<br>
3) [replace_after]<br>
4) Number increment/decrement<br>
5) Input string<br>
